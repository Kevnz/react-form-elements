# Color Input Component

Color Input to be used with `react-form-elements`

Basic usage:

```jsx static
import React from 'react'
import { ColorInput } from 'react-form-elements'

<ColorInput
  label="Color"
  name="color"
  initialValue="#0668fa"
/>

```
