# CheckBox Component

CheckBox to be used with `react-form-elements`

Basic usage:

```jsx static
import React from 'react'
import { CheckBox } from 'react-form-elements'
  <Checkbox
    label="My Checkbox"
    isChecked={true}
    name="check"
    value="checkpoint"
  />
  // if the checkbox is checked the value will be returned, otherwise null
```

